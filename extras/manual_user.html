Manual de Usuario - Sistema de Archivo de Pacientes

Introducción

El Sistema de Archivo de Pacientes es una aplicación web responsiva diseñada para simplificar la gestión de un archivo físico de pacientes. Permite registrar y buscar pacientes de manera eficiente utilizando su DNI y la ubicación de la caja donde se encuentra su expediente.

Características Principales

Interfaz móvil: Diseño optimizado para dispositivos móviles.
Registro de pacientes: Asocia un DNI con un número de caja específico.
Búsqueda rápida: Encuentra rápidamente la caja de un paciente por su DNI.
Importación CSV: Carga masiva de pacientes desde un archivo CSV.
Validación de datos: Verificación de DNI y prevención de duplicados.
Estadísticas: Visualización de totales de pacientes y cajas utilizadas.
Gestión completa de DNI: Listar, editar, mover y eliminar registros.
Gestión de Cajas: Visualizar pacientes por caja y renombrar cajas.
Exportación de datos: Guarda los registros en formato CSV.

Uso del Sistema

1. Registrar Paciente

Sigue estos pasos para registrar un nuevo paciente en el sistema:

  1. Navega a la sección de **Registro de Pacientes**.
  2. Ingresa el DNI del paciente (solo números, entre 7 y 8 dígitos).
  3. Ingresa el Número de Caja donde se archivará el paciente (ejemplo: A-001, B-002).
  4. Confirma para guardar la información.

2. Buscar Paciente

Para encontrar la ubicación de la caja de un paciente:

  1. Dirígete a la sección de **Búsqueda de Pacientes**.
  2. Ingresa el DNI del paciente que deseas buscar.
  3. Inicia la búsqueda.
  4. El sistema te mostrará la caja donde está archivado el paciente.

3. Importar CSV (Carga Masiva)

Puedes importar múltiples pacientes a la vez desde un archivo CSV:

  1. Accede a la función de **Importar CSV**.
  2. Asegúrate de que tu archivo CSV tenga el siguiente formato:
    DNI,Caja
    12345678,A-001
    87654321,B-002
  3. Selecciona el archivo CSV desde tu computadora.
  4. Procede con la importación.

4. Listar y Gestionar Pacientes

Para ver y administrar la lista completa de pacientes:

  1. Ve al **Listado de Pacientes**.
  2. Usa el campo de búsqueda para filtrar pacientes por DNI o número de caja.
  3. Utiliza las opciones "Ordenar por" y "Orden" para organizar la lista.
  4. Desde esta lista, puedes realizar las siguientes acciones por cada paciente:
    • Editar Paciente: Para cambiar el DNI o la caja de un paciente.
    • Mover Paciente de Caja: Para asignar un paciente a una nueva caja.
    • Eliminar Paciente: Para borrar permanentemente un registro de paciente. Se te pedirá confirmación.

5. Gestionar Cajas

Para administrar las cajas y sus contenidos:

  1. Accede a la sección de **Gestión de Cajas**.
  2. Verás un listado de las cajas existentes.
  3. Puedes hacer clic en el nombre de una caja para ver todos los pacientes archivados en ella.
  4. Renombrar Caja: Puedes cambiar el nombre de una caja existente.

6. Exportar Datos

Puedes exportar todos los datos de los pacientes a un archivo CSV para guardarlos o utilizarlos en otras aplicaciones:

  1. Utiliza la función de **Exportar Datos**.
  2. El sistema generará un archivo CSV con todos los registros de pacientes y lo descargará en tu dispositivo.

Problemas Comunes

Si encuentras algún problema o tienes dudas al usar el sistema, considera los siguientes puntos:

  • Verifica que hayas ingresado correctamente los datos (DNI y número de caja).
  • Para la importación CSV, asegúrate de que el formato del archivo sea el correcto.
